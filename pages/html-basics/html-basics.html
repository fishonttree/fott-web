The target attribute points where the linked resource will be opened. Without it, the default value is self, thereby opening in the current tab. Putting _blank as the value will open the link in a new window.

Absolute links take the protocol://domain/path structure.

There's an image below. Image elements are self-closing, and take the src (source) attribute to retrieve where the image file is located.

An image can also be picked from the directory where the files are held.

Images must also have an alt (alternative) attribute for describing it in the event that the reader somehow can't read it.
